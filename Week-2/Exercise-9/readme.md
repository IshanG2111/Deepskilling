# Exercise 9

Spring Boot REST application for the library management system.

## Run it
From the `Week-2/Exercise-9` folder, run:

```powershell
mvn spring-boot:run
```

## Endpoints

```text
GET  /api/books
GET  /api/books/{id}
POST /api/books
```

The application starts with sample books loaded into the in-memory H2 database.
